나만의 문제 해결

  • API 요청 시 GET 메서드를 사용하여 헤더를 설정하고, 응답 코드에 따라 성공과 오류를 구분하여 처리.
    이로써 API 호출의 성공 여부를 정확히 확인하고, 적절한 응답을 반환 가능.
  • JSON 데이터를 파싱하여 뉴스 아이템을 리스트로 변환하고, Gson 라이브러리를 사용하여 보기 좋게 포맷팅.
    이로써 JSON 데이터를 구조화하여 쉽게 읽고 이해할 수 있게 변환.
