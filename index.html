KampingMate

캠핑장 추천&캠핑장 관련정보 제공 페이지

  • 캠핑 초보자가 사용가능한 웹사이트를 제공하자 !
  • 초심자를 위한 가이드 페이지 제공
  • 캠핑장에 대한 정보뿐 아니라 캠핑에 대한 연관정보까지 제공

나만의 문제 해결

  • API 요청 시 GET 메서드를 사용하여 헤더를 설정하고, 응답 코드에 따라 성공과 오류를 구분하여 처리.
    이로써 API 호출의 성공 여부를 정확히 확인하고, 적절한 응답을 반환 가능.
  • JSON 데이터를 파싱하여 뉴스 아이템을 리스트로 변환하고, Gson 라이브러리를 사용하여 보기 좋게 포맷팅.
    이로써 JSON 데이터를 구조화하여 쉽게 읽고 이해할 수 있게 변환.

FoodMate

음식메뉴 추천 페이지

  • 음식도 추천받고,
  • 건강하게 직접 해 먹을 수 있는 사이트를 만들자!
  • 개인 바디데이터와 사용자가 선택한 음식 추천 조건을 선택

나만의 문제 해결

  • 사용자 정보를 세션에서 가져와 새 문의에 필요한 사용자 정보를 자동으로 설정하고 저장.
    이로써 사용자가 별도로 자신의 정보를 입력할 필요 없이, 시스템에서 자동으로 사용자 정보를 처리
  • 로그인된 사용자의 이름과 문의 목록을 동적으로 모델에 추가하여 뷰에 전달. 이를 통해 사용자 맞춤형 화면을 제공하며,
    사용자가 자신이 작성한 문의 목록을 쉽게 확인 가능.
